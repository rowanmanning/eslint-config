
# ESLint Config

A shared [ESLint](https://eslint.org/) config for my open source projects.


## Usage

Install with:

```sh
npm install --save-dev @rowanmanning/eslint-config
```

Add one of the following to your `.eslintrc` file:

```json
{
    "extends": "@rowanmanning/eslint-config"
}
```
